#Instagram Clone
This project was set by our coach Alex Peattie during weeks 9 and 10 at Makers Academy. The idea was for us to create a clone of the popular website Instagram using the Rails environment in order to gain greater familiarity with it.

##Technologies Used
- Sublime Text
- Ruby
- Git
- Ruby on Rails
- Rspec
- Devise
- Paperclip
- Amazon Web Services
- Bootstrap
- jBuilder
- Javascript
- AJAX
- HTML & CSS
